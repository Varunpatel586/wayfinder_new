
## üîß Step 1: Add Missing Columns

Open SQLite in Git Bash:

```bash
cd /e/CODES/pbl/wayfinder_new/data
sqlite3 users.db
```

Then run these ALTER TABLE commands:

```sql
-- Add missing columns
ALTER TABLE places ADD COLUMN category TEXT;
ALTER TABLE places ADD COLUMN country TEXT;

-- Verify columns were added
.schema places
```

You should now see `category` and `country` in the output.
